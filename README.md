## About

Object-Oriented Programming is a powerful paradigm that models software around real-world entities using classes and objects. This repository demonstrates core OOP principles in Java, including:

- **Encapsulation**
- **Inheritance**
- **Polymorphism**
- **Abstraction**

Each topic is explained with well-documented Java code, making it easy to understand and adapt for your learning or teaching purposes.

## Getting Started

To run the code examples in this repository:

1. **Clone the repository**
   ```bash
   git clone https://github.com/shreyansh30/OOP-in-Java.git
   cd OOP-in-Java
   ```

2. **Open in your IDE**
   - You can use **IntelliJ IDEA**, **Eclipse**, **VS Code**, or any Java-supporting IDE.

3. **Compile and Run**
   - Compile and run the `.java` files individually, or configure a project to run multiple examples.

   ```bash
   javac src/SomeExample.java
   java src.SomeExample
   ```
